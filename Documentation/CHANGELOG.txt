## [2.0.0] — 2026-06-16

### Added
- Full 4-layer architecture: Client → Integration → Cognitive → Storage
- FastAPI REST Gateway (`server.py`) with sliding-window rate limiter
- Versioned API endpoints: `/v1/chat`, `/v1/chat/stream`, `/v1/engines`, `/v1/health`, `/v1/status`
- Admin endpoints: `/v1/admin/sessions` with secret-key protection
- Lazy Engine Registry — engines loaded on first request only
- Session TTL Manager — auto-eviction after configurable idle period (default 3600 s)
- Isolated Memory Service microservice (`memory_service.py`, port 8081)
- 5-Dimensional Emotion Engine (`core/emotion_engine.py`)
- Cognitive Monologue Stream (`core/cognitive_stream.py`) — FIFO queue (Impulse → Reflection → Conflict)
- Relationship Engine (`core/relationship_engine.py`) — 4-stage intimacy progression
- Intent Analyzer, Meta-Cognition Module, NamoNexus Fusion Engine, ASI Simulation Engine
- RAG Memory System — FAISS/Qdrant, 100–150 token micro-chunking
- Decoupled adapter layer: emotion, memory, TTS
- ElevenLabs TTS integration
- Telegram Bot with Whisper STT and DALL-E 3 image generation
- HTML5 Web Client with SSE streaming support
- Docker support with Cloud Run deployment scripts
- Usage logging (JSONL) for analytics and billing pipeline
- API key plan system: `public`, `creator`, `studio`
- MIT License under NamoNexus — Kanin Raksaraj
- Full documentation: README, ARCHITECTURE, API_SPEC, INSTALL_GUIDE, COMMERCIAL_STRATEGY
